# Food Ordering App

A food ordering application built with **Next.js**, **shadcn UI**, and **Zustand**.

## Features
- Browse menu items
- Add/remove items from cart
- Checkout with delivery details
- Place multiple orders
- Track order status via modal with simulated real-time updates

## Tech Stack
- Next.js (App Router)
- TypeScript
- shadcn/ui
- Tailwind CSS
- Zustand

## Running Locally

```bash
npm install
npm run dev
